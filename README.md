# Cook County Property Appraisal — Random Forest Valuation Model

This project builds a residential property valuation model using real-world Cook County data under strict runtime and methodological constraints. The objective was to balance predictive accuracy and computational efficiency without exceeding a 30-minute execution limit.

## Project Scope

- 100% independently designed and executed  
- Built Random Forest valuation model (200 trees)  
- Implemented target encoding for high-cardinality variables  
- Engineered interaction and derived features  
- Performed price-segment RMSE error analysis  
- Ranked 4th out of 40+ graduate teams  

## Core Competencies Demonstrated

- Model design under computational constraints  
- Feature engineering with structured real-world data  
- Target encoding for large categorical spaces  
- Random Forest implementation in R  
- Segment-level error diagnostics  
- Accuracy–efficiency trade-off evaluation
