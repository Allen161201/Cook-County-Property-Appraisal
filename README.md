# Cook County Property Appraisal — Random Forest Valuation Model

This project builds a residential property valuation model using public real-world Cook County property transaction data under strict runtime and methodological constraints. The objective was to balance predictive accuracy and computational efficiency without exceeding a 30-minute execution limit.

## Project Scope

- 100% independently designed and executed  
- Built Random Forest valuation model (200 trees)  
- Implemented target encoding for high-cardinality variables (845 neighborhoods, 476 school districts)  
- Engineered interaction and derived features  
- Compared LASSO vs Random Forest performance  
- Ranked 4th out of 40+ graduate teams under runtime constraint  

## Core Competencies Demonstrated

- Applied machine learning under computational constraints  
- High-cardinality categorical encoding strategy  
- Feature interaction engineering  
- Model selection tradeoff analysis (interpretability vs performance)  
- Segment-level RMSE error diagnostics  
- Structured predictive performance evaluation
